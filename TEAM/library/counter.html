Instruction on how to use the counter

  1. Copy the javascript under page settings
  2. Make sure the number that is counting has .counter-value class (number that is shown in the webflow editor is the starting count number)
  3. Open Custom Attributes, enter the desired final counting number in this following syntax fashion: data-count="300"
  4. Put #counter ID in any element for counting trigger
